# MINI-ROCKET-LEAGUE-CSE423Project
A two-player football game built using OpenGL and Python.

## Description
This is a two-player football-style game developed using Python and OpenGL.  
Players control cars to score goals within a fixed time limit.

## Controls
### Player 1 (Red Team)
- W / A / S / D – Movement

### Player 2 (Blue Team)
- Arrow Keys – Movement

## Game Features
- Rectangular football field with goal posts and nets
- Toss system (Head/Tail)
- Team selection (Red / Blue)
- Attacker and defender roles
- Fixed defender movement in front of goal
- Goal detection and score update
- Goal celebration animation (poppers)
- Timer-based gameplay
- 

## Technologies Used
- Python
- PyOpenGL
- GLUT
